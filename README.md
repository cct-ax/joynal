# jornal

新人研修の日報管理Webアプリです。

## 概要

新人が日報を入力・閲覧し、メンター・OJTが週次コメントを行い、管理者がユーザーや権限を管理するWebアプリです。

## 技術スタック

| 項目 | 技術 |
|------|------|
| フロントエンド | Nuxt 4 |
| バックエンド/DB | Supabase |
| ホスティング | Vercel |

## ユーザー種別

| 種別 | できること |
|------|-----------|
| 新人 | 自分の日報を入力・編集・閲覧 |
| メンター | 担当新人の日報閲覧、週次コメント入力 |
| OJT | 担当新人の日報閲覧、週次コメント入力 |
| 管理者 | 全日報閲覧、ユーザー管理、権限設定 |
